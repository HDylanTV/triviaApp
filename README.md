# Trivia Application

A trivia application using https://opentdb.com/api_config.php as source

## Description

Using Vite + React + Typescript as the frontend, and quarkus as the backend.

## Getting Started

### Dependencies

* Docker
* a browser

### Installing

* The frontend and backend can be started up using docker compose.

### Executing program

Start the frontend and backend by running the following command in the root of the project:
```
docker compose up --build
```

then, using the browser go to localhost:5173
